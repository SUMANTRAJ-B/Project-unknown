This project presents an image-based vehicle classification system that uses deep learning to categorize vehicles into five classes: car, bike, bus, truck, and ambulance. A Convolutional Neural Network (CNN) with transfer learning is used to extract features and achieve high accuracy on real-world images.

The system includes an intelligent decision layer that evaluates prediction confidence and classifies outputs into high confidence, needs review, or uncertain categories. Special emphasis is given to ambulance detection by enhancing feature recognition and improving dataset balance.

Designed for real-world applications, this system can support smart traffic management by identifying emergency vehicles and enabling priority-based decision-making.
